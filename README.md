### Steam download monitor

Консольный монитор активности Steam: пытается определить, какая игра/приложение сейчас обновляется/скачивается, показывает статус и текущую скорость.

Работает в Windows

1) Находит папку Steam через реестр Windows.
2) Считывает список библиотек Steam из `steamapps/libraryfolders.vdf`.
3) Берёт хвост лога `logs/content_log.txt` и определяет:
   - активный AppID,
   - статус (Загрузка/Пауза/Проверка/...),
   - скорость загрузки (если в логе присутствует `Current download rate`).
4) По `steamapps/appmanifest_<AppID>.acf` вытаскивает имя игры.
5) Обновляет вывод в консоли с заданным интервалом.

### Запуск

Требования: Python 3.8+

#### 5 обновлений с интервалом 60 секунд (по умолчанию)

```bash
python steam_monitor.py
```

#### Непрерывно

```python
python steam_monitor.py --watch
```

#### 10 обновлений с интервалом 5 секунд

```
python steam_monitor.py --ticks 10 --interval 5
```

#### Непрерывно с ежесекундным тиком

```
python steam_monitor.py --watch --interval 1
```

При смене игры на загрузку может потребоваться некоторое время на обновление статуса, так как лог стима пишется иногда с задержкой.
